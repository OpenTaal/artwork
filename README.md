Gebruik waar mogelijk de SVG-bestanden. Indien niet anders kan, gebruik dan
de PNG-bestanden. De lichtblauwe steunkleur is `#9bcdff`. Het kleurverloop
eindigt in `#f4faff` dat bijna wit is.
